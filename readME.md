# Disease Simulation
## Authors: Jonathan Salazar and Cyrus McCormick

### Configuration file
 Configuration file name is provided via CLA,
 parameters are in the following order:
 1. Dimensions (r, c)
 2. Exposure distance
 3. Incubation period
 4. Sickness time
 5. Recovery probability
 6. Initial agent locations
 7. Initial sick
 8. Number of agents
 9. Initial agent location
 10. Initial sick
